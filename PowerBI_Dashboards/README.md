# Power BI Dashboards

This folder contains interactive Power BI dashboards demonstrating data visualization and business intelligence capabilities.

## üìä Dashboards

### 1. **Bank Loans Dashboard** (`Bank_Loans_Dashboard.pbix`)
Comprehensive loan portfolio analysis dashboard featuring:
- **Loan Portfolio Overview**: Total loans, active loans, and portfolio distribution
- **Risk Assessment**: Loan status breakdown, default rates, and risk categorization
- **Performance Metrics**: Payment trends, collection rates, and portfolio health
- **Geographic Analysis**: Loan distribution by region/location
- **Time Series Analysis**: Loan performance over time

**Files:**
- `Bank_Loans_Dashboard.pbix` - Interactive Power BI file
- `Bank_Loans_Dashboard.pdf` - Static PDF export

**Key Features:**
- Interactive filters and slicers
- Drill-down capabilities
- Real-time data refresh
- Custom visualizations

---

### 2. **Sales Summary Dashboard** (`Sales_Summary_Dashboard.pbix`)
Sales performance tracking and analysis dashboard featuring:
- **Revenue Analysis**: Total sales, revenue trends, and growth metrics
- **Product Performance**: Top-selling products, product categories, and sales distribution
- **Customer Insights**: Customer segmentation, purchase patterns, and customer lifetime value
- **Regional Performance**: Sales by region, territory analysis, and geographic trends
- **Time-based Analysis**: Monthly/quarterly sales trends, seasonal patterns

**Files:**
- `Sales_Summary_Dashboard.pbix` - Interactive Power BI file
- `Sales_Summary_Dashboard.pdf` - Static PDF export

**Key Features:**
- KPI cards and metrics
- Comparative analysis
- Trend visualizations
- Export capabilities

## üöÄ How to Use

1. **Open Power BI Desktop**
2. **Open the `.pbix` file** you want to explore
3. **Interact with the dashboard** using filters, slicers, and drill-down features
4. **Export reports** as PDF if needed

**Note:** Power BI Desktop is required to open `.pbix` files. You can download it from [Microsoft Power BI](https://powerbi.microsoft.com/desktop/).
